# React-Boilerplate

> `React` 项目脚手架

## 介绍

## 目录说明

- `app/` 存放源代码
- `build/` 存放脚手架工具
- `dist/` 存放打包后的项目文件
- `server/` 服务端渲染

## Build Setup

```bash
# 全局安装
#
npm i -g webpack webpack-cli babel-cli

# 安装本地依赖
#
npm i

# 运行 API 服务器
#
npm run backend

# 运行 React-Boilerplate 实例
#
npm run dev-client
```
